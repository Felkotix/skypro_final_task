Git

  • git status - показывает состояние всех файлов в репозитории.
  • cd - расшифровывается как "Change directory", что переводится как "сменить папку" или "перейти в другую папку".
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • git init - создаёт новый локальный репозиторий с заданным имене.
  • git push - загружает все изменения локальной ветки в удаленном репозитор.
  • git add - индексирует указанный файл для последующего коммита, для подготовки изменений в файлах к следующему коммиту, добавляет их из рабочего каталога в так называемую "staging area" (индекс). Это первый шаг в процессе сохранения изменений.
  • git commit -m "[сообщение с описанием]" - фиксирует проиндексированные изменения и сохраняет их в истории.
  • git rm - удаляет конкретный файл из рабочей директории и индексирует его удаление.
